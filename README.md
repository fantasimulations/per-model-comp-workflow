# (Fantasimualtions') Personal Model Comparison Workflow

The big labs and a bunch of others make very useful, interesting and sometimes scary benchmarks and comparisons of LLMs.

I do not think I can do better.  But apart from a starting point, they are useless to me, as I want to run such tests on my own hardware, to understand what I am using.  This workflow is an attempt to make a repeatable test that respects a few small parameters, and fits my needs.

In short, it calls an Ollama server, gets a list of models there, cuts down the list to models that it wants to test, and then sends the models a series of questions.  The same questions, same order, to all the models on that server.
In the future, I will have it limit the questions to a set of tags taht are chosen/appropriate, poossibly to a set of models.  
But I am a right bugger for scope creep, and not finishing.
